# Changelog

## [1.1.0] — 2026-03-11

### Добавлено
- 🔢 `-g, --generate N` — генерация нескольких вариантов на выбор (до 5)
- 📦 `-a, --all` — автоматически `git add .` перед генерацией
- 📋 `-c, --clipboard` — скопировать в буфер без коммита
- ✅ `-y, --yes` — авто-подтверждение без вопросов
- 🚫 `-x, --exclude <file>` — исключить файл из анализа
- 😀 `--gitmoji` — добавить gitmoji к сообщению
- 🛠️ `setup` — интерактивная настройка (режим, язык, провайдер)
- 📊 `stats` — статистика твоих коммитов с графиком
- Настройки сохраняются в `~/.git-vibes.json`
- Статистика сохраняется в `~/.git-vibes-stats.json`

## [1.0.0] — 2026-03-11

### Первый релиз 🎉
- 🧠 Умный режим — конкретные сообщения на русском
- 😤 `--honest` — честный и саркастичный
- 🏢 `--corporate` — корпоратные buzzwords
- 🌸 `--poetic` — поэтический стиль
- 🌀 `--chaos` — полный хаос и рандом
- 🔥 `--roast` — сначала зароастит код, потом даст сообщение
- 🇬🇧 `--lang en` — Conventional Commits на английском
- 🪝 `--install-hook` — автозапуск при каждом `git commit`
- Работает **без API ключа** через `claude` CLI (Claude Code / Max)
- Интерактивный режим: подтвердить / отменить / отредактировать
